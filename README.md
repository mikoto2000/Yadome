Yadome
======

Yet Another DOM Editor.


Usage:
------

```sh
java -jar Yadome-x.x.jar
```


Requirements:
-------------

- java version "1.8.0_112" or later.


Feature :
---------

- [ ] : View DOM tree in tree view.
    - [ ] : Design WIP.
- [ ] : Edit DOM tree in tree view.
    - [ ] : Design WIP.
